LeKiwi 现成玩法总览：能拿它做什么 The LeKiwi playbook — what this mobile-manipulator can actually do out of the box

2026-07-18 · 依据 LeRobot 官方 LeKiwi 文档 + SIGRobotics-UIUC/LeKiwi 仓库，并对齐本板实测（lerobot 0.5.2 @26ff40d，9 电机在线）

TL;DR — 五种玩法，一条主线

  • LeKiwi 的全部玩法都围绕 LeRobot 的模仿学习闭环：遥操作 → 录数据 → 训练策略 → 自主执行。每一步都有现成命令，不用自己写控制器。
  • ① 遥操作：主臂示教机械臂 + 键盘 WASD 开底盘（我们的方案改成手柄）。这是最先能玩的，不需要训练。
  • ② 录数据集：一边遥操作一边录关节+双相机，自动存成 LeRobot 格式、可上传 HuggingFace Hub 共享。
  • ③ 训练策略：ACT / SmolVLA / Diffusion / π0 等 14 类策略任选，云端 GPU 跑（Orin 目前 torch 无 GPU，训练在云上）。
  • ④ 自主执行 & 回放：让训好的策略自己开车抓取（evaluate/rollout），或原样回放录过的一段（replay）。
  • ⑤ 现成资产：Hub 上有社区共享的 LeKiwi/LeRobot 数据集与预训练策略，可以直接下来跑，不必从零采数据。
  • 经典入门任务：移动到不同位置抓取物体放进筐里（官方推荐 ≥50 段、每个位置 10 段）。

玩法全景The whole loop at a glance

① 遥操作：不训练就能玩Teleoperation 难度：入门

LeKiwi 是双机结构：机器人本体跑 lekiwi_host（我们的方案里 host 和客户端都在 Orin，官方是 Pi + 笔记本）。它同时接收两路输入：

  • 机械臂：移动主臂（Leader），从臂 1:1 跟随（示教）。我们的方案用手柄摇杆替代主臂，见 docs/lekiwi-orin-teleop-plan.html。
  • 底盘：官方用键盘。三档速度，按 R/F 加减速：
按键动作速度档线速度转速
W / S前进 / 后退Fast0.4 m/s90 °/s
A / D左移 / 右移（平移，不转身）Medium0.25 m/s60 °/s
Z / X左转 / 右转Slow0.1 m/s30 °/s
# 机器人本体（Orin 上）
python -m lerobot.robots.lekiwi.lekiwi_host --robot.id=my_awesome_kiwi
# 客户端（改 examples/lekiwi/teleoperate.py 里的 remote_ip；本机调试填 127.0.0.1）
python examples/lekiwi/teleoperate.py
⚠ 官方键盘控底盘依赖全局键盘后端（pynput）只在 X11 / Windows / macOS（授权 Input Monitoring）下能用，Wayland 和无头 SSH 下失效。这正是我们改用手柄（pygame joystick）的原因之一 —— 手柄不受这个限制，SSH 无头也能跑。

② 录数据集：把演示存下来Record a dataset 难度：入门

一边遥操作一边录：每一帧同时记录 9 个关节角 + 前视相机 + 腕部相机，自动存成 LeRobot 标准格式（~/.cache/huggingface/lerobot/{repo-id}），可选一键上传 HuggingFace Hub 供社区复现。

python examples/lekiwi/record.py   # 先改脚本里的 remote_ip / repo_id / task
录制中按键作用
→ 右方向键 / n提前结束当前段，进入下一段
← 左方向键 / r重录当前段
Esc / q结束录制（别按 Ctrl+C，会损坏工程文件）
✔ 什么是「好数据集」入门任务选「抓取物体放进筐」：官方建议 ≥50 段、每个位置 10 段，相机固定、抓取动作一致、被操作物在画面内。经验法则：只看相机画面你自己也能完成这个任务，策略才学得会。先别急着加变化，等抓取稳了再逐步引入不同位置/手法。

③ 训练策略：让它学会自己做Train a policy 难度：进阶（需云端 GPU）

录够数据就能训练。LeRobot 内置 14 类策略，命令统一是 lerobot-train --policy.type=<名>：

策略是什么适用
actAction Chunking Transformer首选入门，模仿学习标配，10 段数据 RTX5090 约 2h
diffusionDiffusion Policy多模态动作分布
smolvla / pi0 / pi05视觉-语言-动作大模型（VLA）可跟语言指令，需额外 pip install -e ".[smolvla]"
tdmpc / vqbet / sac其它模仿/强化学习策略研究对比用
lerobot-train \
  --policy.type=act \
  --dataset.repo_id=${HF_USER}/lekiwi_pick \
  --policy.device=cuda \
  --output_dir=outputs/train/kiwi_act \
  --wandb.enable=false
⚠ 训练在云端，不在 Orin 上本板 pip 装的 torch 是 cu130，与 JetPack 6.2.2 的 CUDA 12.6 不匹配，cuda.is_available()==False。训练走云 GPU（维特文档给了 AutoDL 全流程，2 小时级）；训好的 model.safetensors 下回 Orin 做推理。等做推理时再解决 Jetson 官方 torch + 开 Super 模式。

④ 自主执行 & 回放Autonomous eval & replay 难度：进阶

玩法命令要不要训练做什么
评测策略examples/lekiwi/evaluate.py要加载训好的策略，让机器人自己开车+抓取，同时录评测数据
Rolloutexamples/lekiwi/rollout.py要跑策略推理循环（评测的底层形式）
回放examples/lekiwi/replay.py不要把录过的某一段动作原样重放，验证机械重复性 —— 遥操作后马上能玩

先 lekiwi_host 起本体，再在客户端跑上面的脚本（改 remote_ip / 模型路径 / 段号）。

⑤ 复用社区资产：跳过采数据Reuse community assets 难度：入门

不想从零采 50 段数据，可以直接玩别人的：HuggingFace Hub 上按 LeRobot 标签 搜，有大量社区数据集和预训练策略。下载别人的数据集就能本地训练，或下预训练策略直接 evaluate。可视化数据集用：

lerobot-dataset-viz --repo-id <某个社区数据集>   # 浏览器里看关节曲线+相机回放

对我们这台车：从哪开始Where we are, what's next

✔ 已就位9 电机单总线全在线（臂 1~6 位置模式、轮 7/8/9 速度模式）；lerobot 0.5.2 装好并实测读通；底盘全向运动学脚本 base_move.py 已与 lerobot 源码交叉验证零误差；手动试车 notebook notebooks/lekiwi-manual-test.ipynb 可点。

按玩法难度，建议路径：

  1. 先辨认轮子（notebook 第一步）—— 确认 ID 7/8/9 对应左/后/右轮，方向对齐。
  2. 标定机械臂：lerobot-calibrate --robot.type=lekiwi --robot.id=orin_kiwi（成品有客服标定 JSON 可直接放进 ~/.cache/.../calibration/ 跳过）。
  3. 手柄遥操作打通（玩法①）—— 我们的方案核心，先能开车能动臂。
  4. 接双相机 → 录一个小数据集（玩法②），跑 replay 验证。
  5. 数据够了上云训 ACT（玩法③）→ 回 Orin evaluate（玩法④）。
